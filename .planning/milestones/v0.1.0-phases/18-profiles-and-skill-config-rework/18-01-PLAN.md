**Import/Profile.lua — new module:**
Create `Import/Profile.lua` with these functions exposed on `ns.Profile`:

1. `ns.Profile.GetSkillConfig()` — returns `ns.db.profiles[ns.db.activeProfile].skillConfig` with fallback to Default if active profile missing. Initializes `profile.skillConfig = {}` if absent.

2. `ns.Profile.SetSkillField(npcID, spellID, field, value)` — writes a single field to the active profile's skillConfig. Creates intermediate tables as needed.

3. `ns.Profile.SwitchProfile(profileName)` — sets `ns.db.activeProfile = profileName`, calls `ns.Import.RestoreAllFromSaved()`, refreshes config UI if open.

4. `ns.Profile.CreateProfile()` — generates next sequential name ("Profile 1", "Profile 2", etc.) by finding max existing numeric suffix + 1. Creates empty profile entry `{ skillConfig = {} }`. Enforces max 15 profiles (returns nil if at limit). Auto-switches to new profile.

5. `ns.Profile.DeleteProfile(name)` — deletes named profile. Returns false if name is "Default". Switches to "Default" after deletion.

6. `ns.Profile.GetProfileNames()` — returns sorted array of profile names (Default always first).

7. `ns.Profile.EncodeProfile(profileName)` — serializes the named profile's skillConfig via AceSerializer:Serialize -> LibDeflate:CompressDeflate -> LibDeflate:EncodeForPrint, prepends "!". Returns encoded string.

8. `ns.Profile.DecodeProfile(str)` — reverse of encode: strip "!", DecodeForPrint, DecompressDeflate, Deserialize. Returns skillConfig table or nil+error.

9. `ns.Profile.ImportProfile(str)` — calls DecodeProfile, creates new sequential profile, sets its skillConfig to decoded data, switches to it. Returns profile name or nil+error.

10. `MAX_PROFILES = 15` local constant.

Library access pattern (top of file):

local AceSerializer = LibStub:GetLibrary("AceSerializer-3.0")

**Import/Pipeline.lua — MergeSkillConfig profile pivot:**

Replace the `ns.db.skillConfig` reference in MergeSkillConfig with profile-aware read:

```lua
local function MergeSkillConfig(npcID, ability, mobClass)
    local profileCfg = ns.db.profiles
        and ns.db.profiles[ns.db.activeProfile]
        and ns.db.profiles[ns.db.activeProfile].skillConfig
    local cfg = profileCfg
        and profileCfg[npcID]
        and profileCfg[npcID][ability.spellID]
    if not cfg then
        if ability.defaultEnabled == false then return nil end
        return {
            name       = ability.name,
            spellID    = ability.spellID,
            mobClass   = mobClass,
            first_cast = nil,
            cooldown   = nil,
            label      = nil,
            ttsMessage = (C_Spell.GetSpellInfo(ability.spellID) or {}).name,
            soundEnabled = false,
        }
    end
    if cfg.enabled == false then return nil end
    local defaultTTS = (C_Spell.GetSpellInfo(ability.spellID) or {}).name
    return {
        name         = ability.name,
        spellID      = ability.spellID,
        mobClass     = mobClass,
        first_cast   = cfg.timed and cfg.first_cast or nil,
        cooldown     = cfg.timed and cfg.cooldown or nil,
        label        = cfg.label ~= nil and cfg.label or nil,
        ttsMessage   = cfg.ttsMessage ~= nil and cfg.ttsMessage or defaultTTS,
        soundKitID   = cfg.soundKitID,
        soundEnabled = cfg.soundEnabled or false,
    }
end
```
