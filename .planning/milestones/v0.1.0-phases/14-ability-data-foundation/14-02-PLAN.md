For each of the 8 dungeons, create a `Data/{DungeonName}.lua` file following the WindrunnerSpire pattern.

**File template (every file starts with):**
```lua
local addonName, ns = ...

ns.AbilityDB = ns.AbilityDB or {}

-- {Dungeon Name} ability database
-- Keyed by npcID; mobClass defaults to WARRIOR, abilities from MDT spells
```

**Extraction algorithm for each MDT file:**
1. Find each enemy entry in `MDT.dungeonEnemies[dungeonIndex]`
2. Extract `["id"]` value (this is the npcID)
3. Extract all numeric keys from `["spells"]` table (these are spellIDs)
4. Skip enemies with no `["spells"]` table or empty `["spells"]` table
5. Create an `ns.AbilityDB[npcID]` entry with `mobClass = "WARRIOR"` and all spellIDs as abilities with `defaultEnabled = false`

**Entry format for each mob with spells:**
```lua
ns.AbilityDB[NPCID] = {
    mobClass = "WARRIOR",
    abilities = {
        { spellID = SPELL1, defaultEnabled = false },
        { spellID = SPELL2, defaultEnabled = false },
    },
}
```
