**Change 1 — Count label on portraits in PopulateList:**

Inside the `PopulateList` function, in the portrait loop (`for p = 1, MAX_PORTRAITS do`), AFTER `GetPortraitTexture(tex, npcIDs[p])` and `tex:Show()`, add count overlay logic:

```lua
-- After tex:Show():
if not tex.countLabel then
    tex.countLabel = row:CreateFontString(nil, "OVERLAY")
    tex.countLabel:SetFont("Fonts\\FRIZQT__.TTF", 10, "OUTLINE")
    tex.countLabel:SetPoint("BOTTOMRIGHT", tex, "BOTTOMRIGHT", 0, 0)
    tex.countLabel:SetTextColor(1, 1, 1, 1)
end
local count = pack.mobCounts and pack.mobCounts[npcIDs[p]] or 1
if count > 1 then
    tex.countLabel:SetText("x" .. count)
    tex.countLabel:Show()
else
    tex.countLabel:Hide()
end
```

In the `else` branch (where `tex:Hide()`), also hide the count label:
```lua
else
    tex:Hide()
    if tex.countLabel then tex.countLabel:Hide() end
end
```

**Change 2 — Config button in footer:**

After the existing `clearBtn` definition (after `clearBtn:SetScript("OnClick", ...)`), add:

```lua
local configBtn = CreateFrame("Button", nil, frame, "GameMenuButtonTemplate")
configBtn:SetSize(80, 22)
configBtn:SetPoint("RIGHT", clearBtn, "LEFT", -8, 0)
configBtn:SetText("Config")
configBtn:SetScript("OnClick", function()
    if ns.ConfigUI and ns.ConfigUI.Toggle then ns.ConfigUI.Toggle() end
end)
```

This places the Config button to the left of the Clear button in the footer row: [Config] [Clear] [Import].
