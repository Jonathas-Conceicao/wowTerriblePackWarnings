  <action>
Modify Display/IconDisplay.lua with three changes:

**1. Update CreateIconSlot signature and add label FontString:**

Change `CreateIconSlot(spellID, duration)` to `CreateIconSlot(spellID, duration, label)`.

After the cooldown sweep block (after `slot.cd = cd` / end of the `if duration` block), add label rendering:

```lua
-- Label text (optional short text on icon bottom edge)
if label and label ~= "" then
    local lbl = slot:CreateFontString(nil, "OVERLAY")
    lbl:SetFont("Fonts\\FRIZQT__.TTF", 9, "OUTLINE")
    lbl:SetPoint("BOTTOM", slot, "BOTTOM", 0, 3)
    lbl:SetText(label)
    lbl:SetTextColor(1, 1, 1, 1)
    slot.label = lbl
end
```

Then add tooltip support (after the label block, before `return slot`):

```lua
-- Tooltip on mouseover
slot.spellID = spellID
slot:EnableMouse(true)
slot:SetScript("OnEnter", function(self)
    GameTooltip:SetOwner(self, "ANCHOR_BOTTOMLEFT")
    GameTooltip:SetSpellByID(self.spellID)
end)
slot:SetScript("OnLeave", function(self)
    GameTooltip:Hide()
end)
```

Use ANCHOR_BOTTOMLEFT because icons are near the top of the screen (ANCHOR_Y = 900).

**2. Update ShowIcon signature:**

Change `function ns.IconDisplay.ShowIcon(instanceKey, spellID, ttsMessage, duration)` to `function ns.IconDisplay.ShowIcon(instanceKey, spellID, ttsMessage, duration, label)`.

Update the CreateIconSlot call inside ShowIcon (line 173) from `CreateIconSlot(spellID, duration)` to `CreateIconSlot(spellID, duration, label)`.

**3. Update ShowStaticIcon signature:**

Change `function ns.IconDisplay.ShowStaticIcon(instanceKey, spellID)` to `function ns.IconDisplay.ShowStaticIcon(instanceKey, spellID, label)`.

Update the CreateIconSlot call inside ShowStaticIcon from `CreateIconSlot(spellID, nil)` to `CreateIconSlot(spellID, nil, label)`.

**4. Add GameTooltip guard in CancelIcon:**

In `CancelIcon`, before `slot:Hide()`, add:

```lua
-- Hide tooltip if it was anchored to this slot
if GameTooltip:GetOwner() == slot then
    GameTooltip:Hide()
end
```

**5. Add GameTooltip guard in CancelAll:**

In `CancelAll`, inside the `for _, slot in ipairs(activeSlots)` loop, before `slot:Hide()`, add the same guard:

```lua
if GameTooltip:GetOwner() == slot then
    GameTooltip:Hide()
end
```
