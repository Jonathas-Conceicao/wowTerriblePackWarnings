**CombatWatcher.lua changes:**

1. **OnCombatStart:** Replace `ns.Scheduler:Start(selectedDungeon, currentPackIndex)` with scanner invocation:
   ```lua
   function CombatWatcher:OnCombatStart()
       if state ~= "ready" then return end

       local dungeon = ns.PackDatabase[selectedDungeon]
       local pack = dungeon and dungeon[currentPackIndex]
       if not pack then return end

       ns.NameplateScanner:Start(pack)
       state = "active"
       if ns.PackUI and ns.PackUI.Refresh then ns.PackUI:Refresh() end
   end
   ```
   Note: Scanner:Start calls Scheduler:StartAbility internally per-mob, so we do NOT call Scheduler:Start here anymore.

2. **OnCombatEnd:** Call scanner Stop BEFORE Scheduler Stop (per Pitfall 5 — scanner must clean up before CancelAll):
   ```lua
   function CombatWatcher:OnCombatEnd()
       if state ~= "active" then return end

       local dungeon = ns.PackDatabase[selectedDungeon]
       local nextIndex = currentPackIndex + 1

       if nextIndex > #dungeon then
           state = "end"
           currentPackIndex = nextIndex
       else
           state = "ready"
           currentPackIndex = nextIndex
       end

       ns.NameplateScanner:Stop()
       ns.Scheduler:Stop()

       if state == "end" then
           print("|cff00ccffTPW|r All packs completed.")
       else
           print("|cff00ccffTPW|r Next: " .. dungeon[currentPackIndex].displayName)
       end
       if ns.PackUI and ns.PackUI.Refresh then ns.PackUI:Refresh() end
   end
   ```

3. **ManualStart:** Update to use scanner instead of Scheduler:Start, so `/tpw start` goes through the same nameplate detection path:
   ```lua
   function CombatWatcher:ManualStart(packIndex)
       if not selectedDungeon then
           print("|cff00ccffTPW|r Error: no dungeon selected. Use /tpw select <dungeon>")
           return
       end

       if packIndex then
           currentPackIndex = packIndex
       end

       local dungeon = ns.PackDatabase[selectedDungeon]
       local pack = dungeon and dungeon[currentPackIndex]
       if not pack then return end

       ns.NameplateScanner:Start(pack)
       state = "active"
       if ns.PackUI and ns.PackUI.Refresh then ns.PackUI:Refresh() end
   end
   ```

4. **Reset:** Add scanner stop before Scheduler stop:
   ```lua
   -- At the top of Reset(), before ns.Scheduler:Stop():
   ns.NameplateScanner:Stop()
   ns.Scheduler:Stop()
   ```
