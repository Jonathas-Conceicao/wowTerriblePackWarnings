<task type="auto">
  <name>Task 1: Create Display/IconDisplay.lua with full icon display API</name>
  <files>Display/IconDisplay.lua</files>
  <action>
Create Display/IconDisplay.lua following the `local addonName, ns = ...` namespace pattern. Expose API on `ns.IconDisplay`.

**Constants (Claude's discretion on values):**
- ICON_SIZE = 40 (square pixel size)
- ICON_PADDING = 4 (gap between icons)
- ANCHOR_X = 8 (left margin from screen edge)
- ANCHOR_Y = -8 (top margin from screen edge)

**State:**
- `activeSlots` — ordered array of visible icon slot frames (for re-layout)
- `slotsByKey` — map of instanceKey -> slot frame (for lookup by key)

**Internal helper: CreateIconSlot(spellID, duration)**
- Create a Frame parented to UIParent, sized ICON_SIZE x ICON_SIZE
- Create a Texture child at "BACKGROUND" layer, SetAllPoints, set texture via `C_Spell.GetSpellTexture(spellID)`
- Add a thin dark border texture (1px EdgeFile or simple backdrop) for non-glow state
- If duration is provided (timed): create a Cooldown child using `CreateFrame("Cooldown", nil, slot, "CooldownFrameTemplate")`, SetAllPoints, SetDrawEdge(true), SetHideCountdownNumbers(false), call SetCooldown(GetTime(), duration)
- Store `slot.cd` reference for later reset
- Return the slot frame

**Internal helper: LayoutSlots()**
- Iterate activeSlots in order
- For each slot at index i: ClearAllPoints(), SetPoint("TOPLEFT", UIParent, "TOPLEFT", ANCHOR_X + (i-1) * (ICON_SIZE + ICON_PADDING), ANCHOR_Y)

**Internal helper: TrySpeak(message)**
- If message is nil, return immediately
- Get voiceID via C_TTSSettings.GetVoiceOptionID(Enum.TtsVoiceType.Standard)
- If nil, fallback: local voices = C_VoiceChat.GetTtsVoices(); if voices and voices[1] then voiceID = voices[1].voiceID end
- If voiceID found: C_VoiceChat.SpeakText(voiceID, message, 0, 100, false)
- IMPORTANT: Use the post-12.0.0 signature (voiceID, text, rate, volume, overlap). Do NOT include a destination parameter.

**Public API:**

`ns.IconDisplay.ShowIcon(instanceKey, spellID, ttsMessage, duration)`
- If slotsByKey[instanceKey] already exists, reset its cooldown: slot.cd:SetCooldown(GetTime(), duration), remove any red glow, return
- Create slot via CreateIconSlot(spellID, duration)
- Store slot.ttsMessage = ttsMessage
- Insert into activeSlots, store in slotsByKey[instanceKey]
- Call LayoutSlots()
- Show the slot

`ns.IconDisplay.ShowStaticIcon(instanceKey, spellID)`
- Guard: if slotsByKey[instanceKey] exists, return (untimed = one icon total, DISP-03/DISP-08)
- Create slot via CreateIconSlot(spellID, nil) — no duration means no cooldown child
- Insert into activeSlots, store in slotsByKey[instanceKey]
- Call LayoutSlots()
- Show the slot

`ns.IconDisplay.SetUrgent(instanceKey)`
- Look up slot from slotsByKey[instanceKey], return if not found
- Create a red border glow using a simple colored texture overlay approach (per user decision — do NOT use LibCustomGlow):
  - Create 4 thin Texture children (top, bottom, left, right edges) colored {1, 0, 0, 1} at "OVERLAY" layer, each 2px thick, positioned along slot edges
  - Store references on slot.glowTextures for later cleanup
  - If glowTextures already exist, just show them (do not re-create)
- Call TrySpeak(slot.ttsMessage)

`ns.IconDisplay.CancelIcon(instanceKey)`
- Look up slot from slotsByKey[instanceKey], return if not found
- Hide the slot frame
- Remove from slotsByKey[instanceKey]
- Remove from activeSlots (find and table.remove)
- Call LayoutSlots() to close gaps

`ns.IconDisplay.CancelAll()`
- Iterate activeSlots, hide each slot frame
- Wipe activeSlots and slotsByKey

**Debug logging:** Use the established `dbg()` pattern with |cff888888TPW-dbg|r prefix for key operations (ShowIcon, CancelIcon, CancelAll, TTS speak).
