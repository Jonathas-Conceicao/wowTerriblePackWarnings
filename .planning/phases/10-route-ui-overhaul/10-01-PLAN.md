  <name>Task 1: Rewrite PackFrame with pull rows, NPC portraits, and header</name>
  <files>UI/PackFrame.lua</files>
  <action>
Full rewrite of UI/PackFrame.lua. Remove the accordion dungeon list pattern entirely (expandedDungeons, DUNGEON_NAMES, header rows with +/- icons). Replace with:

**1. npcIdToDisplayId lookup table** — Built once at file scope by iterating all ns.DungeonEnemies entries:
```lua
local npcIdToDisplayId = {}
for dungeonIdx, enemies in pairs(ns.DungeonEnemies) do
    for _, enemy in pairs(enemies) do
        if enemy.id and enemy.displayId then
            npcIdToDisplayId[enemy.id] = enemy.displayId
        end
    end
end
```

**2. Constants:**
- ROW_HEIGHT = 28
- PORTRAIT_SIZE = 22
- MAX_PORTRAITS = 8
- Frame size stays 300x400

**3. Header** — A FontString below the title bar showing dungeon name + pull count. Use `UpdateHeader()` function:
- If ns.db and ns.db.importedRoute exists: show "DungeonName -- N pulls" in gold (1, 0.82, 0)
- Otherwise: show "No route imported" in grey (0.6, 0.6, 0.6)

**4. Scroll frame** — Same pattern as current (UIPanelScrollFrameTemplate) but TOPLEFT anchored below the header, BOTTOMRIGHT anchored above the footer buttons (leave ~35px at bottom for Import/Clear buttons).

**5. Pull row creation** — CreatePullRow(parent) function returning a Button frame:
- row.bg: background ColorTexture for state highlighting (SetAllPoints)
- row.pullNum: FontString at LEFT, 20px wide, centered, shows pull number
- row.portraits: table of MAX_PORTRAITS Texture objects, each PORTRAIT_SIZE x PORTRAIT_SIZE, anchored LEFT-to-RIGHT starting after pullNum. Each portrait gets `tex:SetMask("Interface\\CHARACTERFRAME\\TempPortraitAlphaMask")` for circular crop. All hidden by default.
- Hover highlight texture (white 0.1 alpha)

**6. Row pool** — `local rows = {}` table. Reuse rows across Refresh calls. Create new rows only when needed.

**7. PopulateList()** — Iterates ns.PackDatabase["imported"]:
- Guard: if ns.PackDatabase["imported"] is nil, show empty state (just header says "No route imported"), hide all rows, return.
- For each pack in packs: get or create a row, set pullNum text, populate portraits from pack.npcIDs using npcIdToDisplayId lookup. For each npcID: if displayId > 0, call SetPortraitTextureFromCreatureDisplayID(tex, displayId); else set fallback texture "Interface\\Icons\\INV_Misc_QuestionMark". Show used portraits, hide unused ones.
- Apply state coloring per row using ns.CombatWatcher:GetState():
  - Same dungeon ("imported") and packIndex == activePack and curState == "active": orange bg (1, 0.5, 0, 0.25)
  - Same dungeon and packIndex == activePack: green bg (0, 1, 0, 0.15)
  - Same dungeon and packIndex < activePack: grey bg (0.2, 0.2, 0.2, 0.3)
  - Otherwise: transparent (0, 0, 0, 0)
- OnClick: call ns.CombatWatcher:SelectPack("imported", packIndex), then ns.PackUI:Refresh()
- Set scrollChild height to total row count * ROW_HEIGHT

**8. Preserve existing patterns:**
- Movable frame with position persistence (ns.db.windowPos)
- Escape to close (tinsert UISpecialFrames)
- RestorePosition function
- Public API: PackUI.Toggle(), PackUI.Show(), PackUI.Hide(), PackUI:Refresh()
- Refresh calls UpdateHeader() then PopulateList()

**Important:** Do NOT create the Import/Clear buttons or popup in this task -- that is Task 2. Leave the bottom 35px of the frame empty for now (the scroll frame BOTTOMRIGHT should be anchored to leave space).
