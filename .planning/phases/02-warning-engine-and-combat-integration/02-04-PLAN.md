```lua
local function ET_Show(text, duration)
    -- Use AddScriptEvent to show a short-lived timeline entry for the alert
    local eventInfo = {
        text = text,
        duration = duration or 5,
    }
    local eventID = C_EncounterTimeline.AddScriptEvent(eventInfo)
    -- Also store for cleanup if needed (using negative counter to avoid bar ID collisions)
    if eventID then
        etEventIDs["alert_" .. GetTime()] = eventID
    end
end
```

Replace the stub DBM_Show function (lines 94-96) with a real DBM implementation. Use DBT:CreateBar for a visual bar (consistent with how DBM /break works):

```lua
local function DBM_Show(text, duration)
    -- Create a short DBM timer bar for the alert
    local barID = "TPW_alert_" .. GetTime()
    DBT:CreateBar(duration or 5, barID)
    -- Set the bar text manually since CreateBar with just duration+id shows the id
    -- Use DBM's built-in approach: the bar text IS the barID for simple bars
    activeBarIDs["alert_" .. GetTime()] = barID
end
```

NOTE: The barID parameter in DBT:CreateBar IS the display text when no localization table is provided. So pass the alert text as the barID directly:

```lua
local function DBM_Show(text, duration)

    DBT:CreateBar(duration or 5, barID)
    activeBarIDs["alert_" .. GetTime()] = barID
end
```
