At file scope, acquire library handles:
```lua

```

Implement `ns.MDTDecode(inString)` following the exact chain from RESEARCH.md Pattern 2:

1. **Validate input:** If `inString` is nil or empty string, return `false, "No input string provided"`.
2. **Strip ! prefix:** Use `inString:gsub("^%!", "")` to get `encoded` and `usesDeflate` count.
3. **Check modern format:** If `usesDeflate ~= 1`, return `false, "Legacy MDT format (no ! prefix) is not supported"`.
4. **DecodeForPrint:** Call `LibDeflate:DecodeForPrint(encoded)`. If nil, return `false, "DecodeForPrint failed -- string may be corrupted or wrong format"`.
5. **DecompressDeflate:** Call `LibDeflate:DecompressDeflate(decoded)`. If nil, return `false, "DecompressDeflate failed"`.
6. **Deserialize:** Call `AceSerializer:Deserialize(decompressed)`. If not success, return `false, "AceSerializer:Deserialize failed: " .. tostring(data)`.
7. **Return success:** `return true, data`.

Do NOT use LibCompress. Do NOT use DecodeForWoWAddonChannel. The function signature is `ns.MDTDecode(inString)` returning `(boolean, table|string)`.
